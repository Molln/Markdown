Markdown 链接

Markdown 链接

一 链接语法格式

[链接名称](链接地址)
或着
<链接地址>

二 示例1

[Molln Github](https://github.com/Molln)

效果

Molln Github

三 示例2 直接使用链接地址

<https://github.com/Molln>

效果

https://github.com/Molln

四 示例3 高级链接

链接也可以使用变量代替，文档末尾附上变量地址即可

示例

[Molln Github][Github]
[Github]: https://github.com/Molln

效果

Molln Github

经测试，在 Github 无效
