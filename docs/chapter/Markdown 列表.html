Markdown 列表

Markdown 列表

Markdown 支持无序列表和有序列表

一 无序列表

无序列表使用 * + - 作为标记

* 条目1
* 条目2
* 条目3

+ 条目1
+ 条目2
+ 条目3

- 条目1
- 条目2
- 条目3

二 有序列表

有序列表使用数字加上 . 号来表示

1. 条目1
2. 条目2
3. 条目3

三 列表嵌套

列表嵌套只需要在子列表中的选项增加四个空格即可

1. 条目1
    - 条目11
    - 条目12
