Markdown 其他

Markdown 其他

一 支持部分 HTML 元素

不在 Markdown 涵盖范围之内的标签，都可以直接在文档里使用 HTML

目前支持的 HTML 元素有：

<kbd> <b> <i> <em> <sup> <sub> <br>...

二 转义

Markdown 使用了很多特殊符号来表示特定的意义，如果需要显示特定的符号则需要使用转义字符，Markdown 使用反斜杠转义特殊字符

\   反斜线
`   反引号
*   星号
_   下划线
{}  花括号
[]  方括号
()  小括号
#   井字号
+   加号
-   减号
.   英文句点
!   感叹号
