# Markdown 链接

## 一 链接语法格式

````markdown
[链接名称](链接地址)
或着
<链接地址>
````



## 二 示例1

````markdown
[Molln Github](https://github.com/Molln)
````

效果

[Molln Github](https://github.com/Molln)



## 三 示例2 直接使用链接地址

````markdown
<https://github.com/Molln>
````

效果

<https://github.com/Molln>



## 四 示例3 高级链接

链接也可以使用变量代替，文档末尾附上变量地址即可

示例

````markdown
[Molln Github][Github]
[Github]: https://github.com/Molln

````

效果

[Molln Github][Github]

[Github]: https://github.com/Molln
