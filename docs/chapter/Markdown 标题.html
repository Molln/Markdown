Markdown 标题有两种格式

一 使用 = 和 - 标记一级标题和二级标题

语法格式

一级标题
======

二级标题
------

显示效果

二 使用 # 标记

使用 # 号可表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推。

语法格式

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
