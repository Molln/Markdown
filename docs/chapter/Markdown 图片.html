Markdown 图片

语法格式

![显示文件](图片链接)
![显示文件](图片链接 "标题")
  • 开头一个感叹号!
  • 方括号，填写图片的替代文字
  • 小括号，填写图片链接地址，可使用引号填写变体

Markdown 暂时无法指定图片的高度与宽度，如果需要，可以使用 <img> 标签
