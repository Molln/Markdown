# Markdown 代码

1 使用反引号 `` `

如果是段落上的一个函数或片段的代码可以使用反引号将其包起来

示例

 ````markdown
`alert()`
 ````

2 使用 **4个空格** 或者 **两个制表符(Tab)**

3 使用 ````包裹一段代码，并指定一种语言(也可以不指定)

````java
​````java
public static void main(String[] args){
    System.out.println("Hello World");
}
​````
````
